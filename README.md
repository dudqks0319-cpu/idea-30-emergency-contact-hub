# 비상연락망 병원 허브

## 개요
- 아이디어 번호: 30
- 한 줄 컨셉: 가족 비상연락망과 근처 병원 정보를 통합
- 수익모델: 구독
- MVP 난이도: 하
- 카테고리: 육아/가정

## 기술 스택
- Flutter (Android/iOS/Web)
- Supabase (Postgres + Auth + RLS)

## 실행
```bash
flutter pub get
flutter analyze
flutter test
flutter run
```
